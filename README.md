# LineCore Portable
LineCore Portable is a reimplementation of LineCore OS in Python.

The goal of LineCore Portable is to create a portable version of LineCore OS that can be run on different operating systems.

LineCore Portable should be on par with version 0.0.57 of LineCore OS.

## Testing

In order to run the tests, use `./all_tests.sh`
You can view the tests at the `./tests` folder.
